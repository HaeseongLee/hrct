# hrct
H-Robot Control Tools

## Block Sorting Task

1. start mujoco simulator.

   
```bash
ros2 run mujoco_ros fr3_block
```

2. start task manager.

   
```bash
ros2 run mujoco_task_planner block_sorting
```
